# Security Policy

## Supported Versions

Currently, only the latest version of AutoTidy is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within AutoTidy, please do not open a public issue. Instead, please report it by emailing Alp Yalay at alpyalay@gmail.com.

All security vulnerabilities will be promptly addressed.
